iAcademics

Organization/Institute:   Indira Gandhi Institute of Technology

Project duration:   September 2014 to May 2015

In this world of smartphones and digitalization, all the activities in our day to day are put online in form of a WebAPP or smartphone app, for handy and easy accessibility. The application, will be serving the teachers, Deans and the students, providing utmost features of time table scheduling, notifications and maintaining the academics records; via their Android smart phones. Time table scheduling refers to the sorting of the time table based on the user’s credentials; iAcademics will give push-notifications for the users, in case of any updating. It will also provide the real time experience of data analysis and keeping track of all academic records. In this we will be using Google Cloud for data storage, Google cloud SQL server acting as the backbone server and the Google Messaging service, for notifications and Instant messaging. Eclipse IDE, with Java as the base language for the App Development will be used for development. This project can further be enhanced to integrate all other modules of academics which includes attendance for both teachers and students; online filing of leave applications for the officials and support all other mobile platforms.
